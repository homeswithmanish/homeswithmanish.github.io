Real-estate conveyancing, run on one system — in any province.

Conveyo automates the closing file: document extraction, jurisdiction-specific checklists, real land transfer tax and rebate calculations, and trust accounting — all in one auditable system of record.

Everything a closing file needs

Purpose-built for conveyancing, not a generic document tool with a real-estate skin.

AI document extraction

OCR and field extraction from tax certificates, instructions, and closing documents — structured data lands in the file, not a PDF pile.

Real tax calculators, not estimates

Land transfer tax, municipal surtax, first-time-buyer rebates, and the GST/HST new-housing rebate — computed from the actual bracket rules for each province, not a rule of thumb.

Jurisdiction-aware checklists

Closing checklists and document templates adapt to the matter's province automatically — Teraview steps in Ontario, LTSA steps in BC, without a manual switch.

Trust accounting built in

Statement of adjustments, trust ledger, and reconciliation in one system — post the balance due on closing straight to the trust account.

ID verification & address validation

Connector integrations for identity/AML checks and address validation, with a clean audit trail on every run.

Built to expand

The jurisdiction engine is modelled as country + region from day one — adding a new province is a configuration change, not a rewrite.

One province today, every province by design

Ontario and British Columbia have real, fully modelled tax and closing rules today. Every other province — and a US state, eventually — is a configuration away, not a re-architecture.
